## Installation

Install it globally with:

```shell
yarn global add @caedes/git-branch-new
```

Upgrade it globally with:

```shell
yarn global upgrade @caedes/git-branch-new
```

Uninstall it globally with:

```shell
yarn global remove @caedes/git-branch-new
```
